## 기능 구현 목록

### 컴퓨터 숫자 생성 -> Class RandomNumber

- 랜덤으로 3자리 자연 수 생성 (0)
- [예외] 자릿수 중복 (0)

### 입력 -> Class UserNumber

- 사용자가 숫자를 입력
- [예외] 숫자이외의값[문자, 공백 줄바꿈 특수문자등] 포함 (0)
- [예외] 숫자가 세자리 수가 아님 (0)
- [예외] 0이 입력 (0)
- 중복된 숫자 입력은 사용자 재량

### 출력 -> Class Result

- 입력한 세자리 수 스트라이크, 불, 낫싱 3가지로 힌트 출력 (0)
- 정답 입력 (0)
- 정답 화면 - 재시작 여부 물어봄
- 게임 종료 여부
- 재시작 1 종료 2
- [예외] 재시작, 종료 값 이외의 숫자 입력

### 동작

- 컴퓨터 1-9 난수 생성 (0)
- [예외] 각 자리 숫자가 달라야함 (0)
- 컴퓨터의 수 - 사용자 입력 수 비교 (0)
- 숫자와 자리가 모두 맞으면 스트라이크 (0)
- 숫자만 맞고 자리가 틀림 불 (0)
- 숫자와 자리 모두 틀림 낫싱 (0)
- 재시작 여부에 따라 재시작하거나 종료한다
